조건문 - if

조건에 따라 실행되는 구문

기본형태 -> if (조건boolean){실행되는 소스코드}

괄호 안의 조건의 결과가 참인 경우 중괄호 안의 소스코드가 실행된다. 조건의 결과가 거짓인 경우는 중괄호 안의 소스코드는 해석되지 않는다.

if(조건){소스코드1}else{소스코드2}

조건이 참이라면 소스코드1을 실행하고, 그렇지 않다면 소스코드2를 실행한다

else구문은 선택적으로 사용하는 것이며, 앞에서 제시한 경우의 정반대 경우를 의미한다(그렇지 않다면으로 해석)

if(조건1){소스코드1}else if(조건2){소스코드2}...

만약 조건1이 참이라면 소스코드1을 실행하고, 그 이후의 조건은 해석하지 않는다.

위에서 아래로 조건을 비교하면서 내려올 때 한번이라도 조건이 참이면 다음부터 해석하지 않는다.
